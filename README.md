## 👨‍💻 **Project** 👨‍💻

O Projeto **Urna-Eletronica** é baseado na criação de uma urna eletrônica simulando uma votação para vereador. Neste projeto só é valido o voto (1,2,3,4) para o vereador "Tiririca".
